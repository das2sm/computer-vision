DETR:
- Gets rid of anchors and NMS
- Bipartite matching, Hungarian Algorithm
- CNN, Transformer
- Different from what's shown in transformer paper as there is no "order" in
  images as there is in language - no masking needed - inference as well as
  training is done in parallel
- Transformer is permutation invariant - fixed positional encodings are needed to know where something is - added to each layer - added to encoder
- Also, different positional embeddings, object queries - learned - to know
  where specific things are in the image - added to input of each layer in decoder
- Disadvantage - slow convergence, improved upon by deformable DETR


Transformers:
- Attention(Q, K, V) is scaled by 1/sqrt(dk) as Q . K makes the variance dk
  instead of 1. To scale the variance back to 1, we divide by sqrt(dk) as
  Var(X / a) = Var(x) / a^2
- Transformer self-drawn architecture
- Positional encoding: different sin/cos values for each dimension - change
  at different speeds - combination creates unique encoding, also helps dete
  rmine relative positions, closer positions are more relative, regardless of actual position in the full thing


DETR3D:
Summary of the Flow:
3D Queries (The "Detectives" with 3D coordinates) -> FIRST
Project to 2D (The "Map" to the camera images)
Collect 2D Features (The "Evidence" from the pixels)
Predict 3D Boxes (The "Verdict") -> LAST

Bilinear Interpolation:
Accurate Feature Alignment: It ensures that the query accurately captures
visual information from the 2D feature maps, even when the projection falls
between pixels (float values), which is essential for accurate 3D object localization.

By using the Homogeneous Counterpart c*li = cli <append> 1 (to match
dimensions when going from 3D to 2D),
clmi = Tm x c*li (clmi - projection of reference point onto m-th camera)
and camera matrices, the model can sample features from multiple cameras simultaneously. If a car is split across the "Front" and "Front-Left" cameras, DETR3D naturally stitches those features together in 3D space.

Uses focal loss and L1 loss
